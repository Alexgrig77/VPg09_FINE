# Скриншоты FISH-MVP

Для сдачи проекта добавьте скриншоты основных экранов:

1. **Логин** — страница входа
2. **Главная** — главная страница после входа
3. **Разрешения** — список разрешений
4. **Рыба** — справочник рыбы
5. **Выловы** — учёт выловов
6. **Статистика** — сводные данные и экспорт

Сохраните файлы в эту папку с понятными именами (например: `01-login.png`, `02-main.png`).
