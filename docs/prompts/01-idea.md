# Промпт 01: Идея и формат проекта

**Этап:** 1.1 Идея  
**Среда:** [ЛОКАЛЬНО]

## Контекст

Подготовка выпускного проекта по курсу. Эталон — проект FISH (база разрешений на рыболовство). Нужно определить идею, формат и цель упрощённого MVP.

## Задача

Создать краткий документ `docs/IDEA.md` с описанием:
- Название проекта
- Идея (что делаем и зачем)
- Формат (веб-приложение)
- Целевая аудитория (1–2 предложения)
- Основная цель проекта

## Критерии приёмки

- Файл `docs/IDEA.md` существует
- Содержит все перечисленные разделы
- Текст лаконичный (до 1 страницы)

## Текст промпта

```
Создай документ docs/IDEA.md для выпускного проекта «База разрешений на рыболовство» (FISH-MVP). 

Включи разделы:
1. Название — FISH-MVP: База разрешений на рыболовство
2. Идея — веб-приложение для учёта разрешений на рыболовство, видов рыбы и выловов (упрощённый MVP аналог полноценной системы FISH)
3. Формат — веб-приложение (Flask + SQLite)
4. Целевая аудитория — малые рыболовные хозяйства, инспекторы, учётчики
5. Цель — создать работающий MVP для сдачи выпускного проекта, демонстрирующий умение разрабатывать веб-приложения с БД и авторизацией

Пиши кратко, без «воды».
```
