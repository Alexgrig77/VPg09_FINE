# Промпт 05: Схема БД и инициализация

**Этап:** 3.1 База данных  
**Среда:** [ЛОКАЛЬНО]

## Контекст

Flask app работает. Нужно создать SQLite БД с таблицами: Разрешения, Рыба, Выловы, Пользователи (проектируем с нуля под предметную область).

## Задача

Скрипт создания схемы и скрипт инициализации (создание папки data, вызов создания таблиц).

## Критерии приёмки

- Запуск init-скрипта создаёт data/FISH.db с таблицами
- Разрешения: Номер_разрешения, Год, Район_промысла, Ответственный, Наименование_групповое, Лимит_кг
- Рыба: Наименование_рыбы, Наименование_групповое, Цена
- Выловы: Разрешение (FK), Дата_вылова, Наименование_рыбы (FK), Количество, Сумма

## Текст промпта

```
Создай схему БД для FISH-MVP:

1. application/models/fish_database.py — класс FishDatabase (работа с сущностями «разрешения / рыба / выловы») с методами:
   - get_connection()
   - create_tables() — создание таблиц Разрешения, Рыба, Выловы
   Упрощённая схема: только ключевые поля, перечисленные ниже.

2. scripts/init_db.py — скрипт инициализации:
   - Создаёт папку data
   - Вызывает create_tables()
   - Печатает "БД инициализирована"

Таблица Разрешения: id, Номер_разрешения, Год, Район_промысла, Ответственный, Наименование_групповое, Лимит_кг, created_at, updated_at.
Таблица Рыба: id, Наименование_рыбы, Наименование_групповое, Цена, created_at, updated_at.
Таблица Выловы: id, Разрешение (TEXT FK), Дата_вылова, Наименование_рыбы (TEXT FK), Количество, Сумма, created_at, updated_at.

Используй DATABASE_PATH из config или os.environ.
```
