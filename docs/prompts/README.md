# Библиотека промптов FISH-MVP

Набор промптов для пошаговой разработки выпускного проекта «База разрешений на рыболовство» в Cursor. Репозиторий на GitHub может называться иначе (например VPg09_FINE) — это не меняет название продукта FISH-MVP.

## Как использовать

1. Открыть файл промпта (`01-idea.md`, `02-spec.md` и т.д.)
2. Скопировать текст из секции **«Текст промпта»** в чат Cursor
3. Запустить выполнение
4. Проверить результат по **«Критериям приёмки»**
5. Подтвердить и перейти к следующему этапу
