# Идея проекта FISH-MVP

## Название

**FISH-MVP: База разрешений на рыболовство**

## Идея

Веб-приложение для учёта разрешений на рыболовство, видов рыбы и выловов.

## Формат

Веб-приложение на Flask + SQLite. Серверная логика, авторизация, REST API, HTML-страницы с формами и таблицами.

## Целевая аудитория

Малые рыболовные хозяйства, инспекторы, учётчики — пользователи, которым нужен простой учёт разрешений и выловов без сложной инфраструктуры.

## Цель

Создать работающий MVP для сдачи выпускного проекта, демонстрирующий умение разрабатывать веб-приложения с БД, авторизацией и деплоем.
